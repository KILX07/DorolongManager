# 🦋 Dorolong Manager (도로롱 매니저)

**Dorolong Manager**는 명조(Wuthering Waves), 젠레스 존 제로(ZZZ), 원신 등 **XXMI,3DMigoto 기반의 모드**를 사용하는 게임을 위해 제작된 간단한 모드 관리 도구입니다.


![Preview Image](https://ifh.cc/v-3bfHD7)


## ✨ 주요 기능 (Key Features)

*   **🔄 자동 새로고침 (Auto Refresh):** 스킨을 켜고 끌 때마다 자동으로 `F10` 키를 입력하여 게임에 즉시 반영합니다.
*   **⌨️ .ini 키 설정 에디터:** 메모장을 열지 않고도 GUI에서 간편하게 토글 키(Key Binding)를 수정할 수 있습니다. (UTF-16, BOM 지원)
*   **📂 드래그 앤 드롭 설치:** ZIP 파일을 프로그램에 끌어다 놓기만 하면 자동으로 설치됩니다.
*   **🎨 다채로운 테마:** Dark, White, Lime, Pink, Forest 등 취향에 맞는 테마와 Glassmorphism 디자인을 지원합니다.
*   **🛡️ 안전한 방식:** 게임 메모리를 건드리지 않고, 파일명 변경(`DISABLED`) 방식을 사용하여 안전합니다.

## 🚀 설치 및 실행 (Installation)

1. **[Releases](https://github.com/KILX07/DorolongManager/releases))** 페이지에서 최신 `DorolongManager`를 다운로드합니다.
2. 설치 후 프로그램을 **관리자 권한**으로 실행합니다. (오버레이 및 키 입력 제어를 위해 필수)
3. **설정(Settings)** 탭에서 게임 실행 파일(`.exe`) 경로를 지정합니다.
4. **모드 그룹**애서 모드 폴더를 연결합니다.

## 🛠️ 개발 환경 (Development)

이 프로젝트는 Python과 Eel(HTML/JS GUI)을 기반으로 제작되었습니다.

### 요구 사항
*   Python 3.10+
*   Chrome or Edge Browser
